# Refactoring Code for Horiseon Webpage

## Description

This project entailed refactoring provided code to title the page, utilize semantic HTML elements, consolidate CSS, and add alt tags to make the webpage accessible.

The page is primarily information about the digital marketing group to scroll though, but the links in the navigation bar take the user to its respective section.
